New Pastebin Features

October 26, 2020

I stumbled across this today while looking for a snippet of code on my Pastebin account.

There are two new features that can be used when creating a new Paste.

  1. An individual password can be assigned to each Paste.
  2. There an option for “Burn after read” that can be enabled for each new Paste.
